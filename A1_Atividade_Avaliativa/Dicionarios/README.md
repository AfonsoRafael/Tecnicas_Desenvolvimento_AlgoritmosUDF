# Dicionários - Cadastro de Produtos

## Descrição da Atividade
Desenvolver um sistema simples de cadastro de produtos com nome e preço, armazenando em dicionário.

## Objetivo
Implementar um sistema de cadastro de produtos utilizando dicionários para armazenar informações de nome e preço de forma organizada.

## Critérios de Avaliação
- **Uso correto de dict()** - Manipulação adequada de dicionários Python
- **Inserção e recuperação de dados** - Capacidade de adicionar e acessar dados no dicionário
- **Estrutura clara e funcional** - Código bem organizado e sistema funcional
