# Estruturas Condicionais - Verificação de Idade

## Descrição da Atividade
Criar um programa em Python que simule um sistema de verificação de idade para entrada em eventos.

## Objetivo
Implementar um sistema que receba a idade do usuário e determine se ele pode entrar no evento baseado em faixas etárias pré-definidas.

## Critérios de Avaliação
- **Uso correto de if, elif, else** - Estrutura condicional bem implementada
- **Entrada e saída de dados** - Interface clara para interação com o usuário
- **Clareza no código e comentários** - Código organizado e bem documentado
