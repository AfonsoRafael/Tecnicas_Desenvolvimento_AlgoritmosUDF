# Listas - Cadastro de Alunos

## Descrição da Atividade
Criar um programa que receba nomes de alunos e armazene em uma lista, permitindo exibir todos os nomes ao final.

## Objetivo
Implementar um sistema de cadastro dinâmico de alunos usando listas, com entrada contínua de dados e exibição final de todos os registros.

## Critérios de Avaliação
- **Uso adequado de listas** - Manipulação correta de operações com listas (append, etc.)
- **Entrada dinâmica de dados** - Sistema que permite adicionar múltiplos alunos de forma interativa
- **Impressão correta dos elementos** - Exibição organizada e completa dos nomes cadastrados
