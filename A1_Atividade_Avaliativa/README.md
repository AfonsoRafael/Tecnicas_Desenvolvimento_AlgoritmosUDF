# Avaliação A1 - Técnicas de Desenvolvimento de Algoritmos

**Data:** 28/11/25  
**Disciplina:** Técnicas de Desenvolvimento de Algoritmos  
**Conteúdos:** Estruturas Condicionais, Estruturas de Repetição, Listas e Dicionários

## Descrição das Atividades

### 1. Estruturas Condicionais
**Tarefa:** Criar um programa em Python que simule um sistema de verificação de idade para entrada em eventos.

### 2. Estruturas de Repetição 
**Tarefa:** Desenvolver um contador de 1 a 100 com `for` e outro com `while`, exibindo apenas números pares.

### 3. Listas
**Tarefa:** Criar um programa que receba nomes de alunos e armazene em uma lista, permitindo exibir todos os nomes ao final.

### 4. Dicionarios
**Tarefa:** Desenvolver um sistema simples de cadastro de produtos com nome e preço, armazenando em dicionário.

## Critérios de Avaliação

- **Estruturas Condicionais (1,0 pt):** Uso correto de `if`, `elif`, `else`; entrada e saída de dados; clareza no código e comentários
- **Estruturas de Repetição (1,0 pt):** Implementação correta dos laços; uso de operadores lógicos; organização do código
- **Listas (1,0 pt):** Uso adequado de listas; entrada dinâmica de dados; impressão correta dos elementos
- **Dicionários (1,0 pt):** Uso correto de `dict()`; inserção e recuperação de dados; estrutura clara e funcional
- **GitHub (1,0 pt):** Repositório público; README explicativo; organização e versionamento com Git
