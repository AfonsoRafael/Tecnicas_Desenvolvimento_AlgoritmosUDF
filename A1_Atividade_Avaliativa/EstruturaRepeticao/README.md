# Estruturas de Repetição - Contador de Números Pares

## Descrição da Atividade
Desenvolver um contador de 1 a 100 com `for` e outro com `while`, exibindo apenas números pares.

## Objetivo
Implementar dois tipos de estruturas de repetição (for e while) para filtrar e exibir números pares dentro do intervalo de 1 a 100.

## Critérios de Avaliação
- **Implementação correta dos laços** - Uso adequado das estruturas for e while
- **Uso de operadores lógicos** - Aplicação correta do operador módulo (%) para identificar números pares
- **Organização do código** - Código limpo, bem estruturado e de fácil compreensão
